# Oplevering Checklist

## Voor development
- [ ] `content/site.ts` ingevuld met klantgegevens
- [ ] Kleuren aangepast in `tailwind.config.ts`
- [ ] Logo/favicon in `public/`
- [ ] `.env.local` geconfigureerd

## Voor oplevering
- [ ] `npm run build` slaagt zonder fouten
- [ ] Alle pagina's getest op mobiel
- [ ] Contactformulier werkt (test email ontvangen)
- [ ] SEO metadata klopt (title, description, OG image)
- [ ] Laadsnelheid < 3 seconden (Lighthouse check)
- [ ] SSL actief
- [ ] Privacy policy pagina aanwezig
- [ ] KVK/BTW nummers correct in footer
- [ ] Alle links werken (geen 404's)
- [ ] Favicons en OG image correct

## Overdracht (eenmalig)
- [ ] GitHub repo getransfereerd naar klant
- [ ] Vercel project getransfereerd
- [ ] Domein overgedragen of DNS ingesteld
- [ ] Documentatie meegeleverd
- [ ] Klant heeft toegang tot alle accounts
- [ ] Laatste factuur verstuurd
